Validador de Contraseñas











Preguntas a responder lab 6

¿Por qué es una buena práctica usar JavaScript para checar que sean válidos los inputs de las formas antes de enviar los datos al servidor?

Porque mejora la experiencia del usuario al darle feedback inmediato, reduce la carga del servidor al no hacerle peticiones inecesarias con datos erroneos y sirve como validación inicial para garantizar seguridad.

¿Cómo puedes saltarte la seguridad de validaciones hechas con JavaScript?

Al deshabilitar javascript como usuario en el navegador, manipulando el codigo de javascript mediante herramientas de desarrollo del navegador o incluse enviando peticiones HTTP alteradas directamentre al servidor evitando así las validaciones del cliente.

Si te puedes saltar la seguridad de las validaciones de JavaScript, entonces ¿por qué la primera pregunta dice que es una buena práctica?

Porque aún puede servir como capa de seguridad, filtra inputs no maliciosos como errores honestos de los usuarios y proporciona una experiencia de usuario más rápida y efectiva.
