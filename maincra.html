Preguntas a responder lab 3

Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

Debido al poder del !important, mi recomendación es solo usarlo cuando sea absolutamente necesario como en situaciones de emergencia o correciones rápidas.

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Por la legibilidad del contenido y los tiempos de carga.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

Usar cada una dependiendo del contexto. Usar "%" si quieres adaptarte al tamaño del contenedor padre. Usar "px" o "pt" para propiedades que no cambian.

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Porque elimina espacios en blanco, comentarios y caracteres innecesarios y asi el archivo CSS que se debe descargar por el navegador es más pequeño.

Preguntas a responder lab 4

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java es un lenguaje de propósito general y JavaScript es un lenguaje de secuancias de comandos para páginas web.

¿Qué métodos tiene el objeto Date?

"getDate()" "getMonth()" "getFullYear()" "getHours()" "getTime()"

¿Qué métodos tienen los arreglos?

"push()" "pop()" "splice()" "slice()" "forEach()"

¿Cómo se declara una variable con alcance local dentro de una función?

Usando "let"

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Puede aumentar la posibilidad de errores ya que diferentes partes del código pueden modificar ese tipo de varibles.
